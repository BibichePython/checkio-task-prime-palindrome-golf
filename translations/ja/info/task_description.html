Prime Palindrome Golf を知っていますか？ 入力としてある数が与えられ、その数より大きく、その数と最も近い対称な素数を探すコードゴルフのことです。

palindromic number もしくは numeral palindrome というのは、各桁を反転しても同じ値になるものです。例えば 79197 のようなものです。これらの数は左右対称です。
素数は 1 より大きい自然数で、かつ 1 と自分自身でのみ割り切れる数のことです。この問題では一つの正の整数が与えられ、次の条件を満足する整数を探します：
- 与えられた数より大きく、
- 素数であり、
- 対称である。
例えば 13 が与えられたとすると、これより大きくて最も近い対称な素数は 101 です。 また、2 が与えられたとすると 3 がその条件を満足します。 なぜなら、一桁の素数は対称だからです。

もう一つルールがあります。 この問題はコードゴルフなので、短いプログラムの方が高い評価を得られます。 スコアはプログラムの文字数で決められます。 250 文字が最大の文字数で 0 点です。 250 文字を基準として 1 文字少なくなるごとに 1 点追加されます。 つまり 200 文字のプログラムは 50 点です。

重要： システムの制約のため、再帰関数のコールは 100 回までです。再帰関数を使わない方法で挑戦してください。

入力： 一つの整数。

出力： 入力値よりも大きい対称な素数の内、最も入力値に近い数。

例：

golf(2) == 3
golf(13) == 101
golf(101) == 131
