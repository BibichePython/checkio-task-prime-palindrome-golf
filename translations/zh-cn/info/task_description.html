你知道关于回文数与素数的游戏吗？ 你的挑战是找出比给定数字大且最接近给定数字的回文素数。

回文数是像79197这样各位数字按相反顺序排列后仍然不变的数字。 这些数字看上去是对称的。
素数是指在大于1的自然数中只能被1与它本身整除的数。 在这个任务中你要做的是找出和给定数字最接近的数，它满足：
- 大于给定数；
- 是素数；
- 是回文数。
例子: 13的大于且最接近它的回文素数是101。 2的答案则是3。因为只有一位的数都是回文数。

这个挑战有一条额外的规则。 这是一个code golf 任务，你的最大目标就是让代码尽可能短。 代码越短得到的点数越多。你所得到的分数直接与代码的长度相关。 作为参考，点数是基于字符数量的。最多只能允许250个字符，而且你将得到0点。 每比250少一个字符，你都赚到1点。比如说200个字符的代码能得到50点。

重要: 请不要尝试用递归来解决这个问题，递归深度限制为100。

输入：一个整数（int）

输出：大于且最接近它的回文素数（int）

范例：

golf(2) == 3
golf(13) == 101
golf(101) == 131

任务目的： CS的许多领域都经常用得到素数。但是，这个任务不是为了优雅的或者是高效率的代码。 这个任务的目的是帮助你缩短代码的那些Python技巧（hack）。 并不是要把这些技巧用在生产中，而是为了更深入地理解Python。

前提： 0 < number < 98689
